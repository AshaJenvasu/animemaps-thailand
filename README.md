# 🎌 Animemaps Thailand 
> The most comprehensive event aggregator for Anime, Cosplay, and Pop-up Stores in Thailand.

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen.svg)(https://ashajenvasu.github.io/animemaps-thailand/)

## 🌟 Overview
**Animemaps Thailand** is a specialized web platform designed to solve the problem of fragmented information regarding anime events in Thailand. Whether it's a major convention like AFA, a local Cosplay gathering, or a limited-time themed cafe, this project brings them all into one unified, easy-to-navigate interface.

This project serves as my **flagship personal project**, demonstrating my journey from core web fundamentals to modern full-stack development.

## 💡 Inspiration
As an avid anime fan who regularly attends events in Thailand, I frequently encountered the problem of having to search for event information scattered across multiple social media pages and websites. It was time-consuming and inefficient. This frustration sparked the idea to build a centralized hub where the community can easily find all upcoming Anime events, Cosplay gatherings, and pop-up stores in one place.

## 🚀 Key Features
- **Dynamic Event Calendar:** Automatically sorted by date to show the most upcoming events first.
- **Smart Filtering:** Filter events by category (Cosplay, Concert, Exhibition) and by month.
- **Responsive Design:** A "Mobile-First" approach ensuring a seamless experience across all devices.
- **Interactive UI:** Features an automated slideshow, search popup, and smooth scroll-to-top functionality.

## 🛠️ Tech Stack (Current: Phase 1)
- **Frontend:** Semantic HTML5, CSS3 (Custom Variables, Flexbox), Vanilla JavaScript (ES6+).
- **Deployment:** GitHub Pages.
- **Version Control:** Git.
